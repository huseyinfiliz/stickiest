# Stickiest

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/huseyinfiliz/stickiest.svg)](https://packagist.org/packages/huseyinfiliz/stickiest) [![Total Downloads](https://img.shields.io/packagist/dt/huseyinfiliz/stickiest.svg)](https://packagist.org/packages/huseyinfiliz/stickiest)

A [Flarum](https://flarum.org) extension. Stick, super stick or tag stick discussions to the top of the list.

## Installation

Install with composer:

```sh
composer require huseyinfiliz/stickiest:"*"
```

## Updating

```sh
composer update huseyinfiliz/stickiest:"*"
php flarum migrate
php flarum cache:clear
```
